Lenguajes de interfaz

El lenguaje de descripción de interfaz o también lenguaje de definición de interfaz (en inglés, interface description language o respectivamente interface definition language, y su acrónimo IDL)

es un lenguaje informático utilizado para describir la interfaz de componentes de software
